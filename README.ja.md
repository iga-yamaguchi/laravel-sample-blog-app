# 保持するデータ

- ユーザーID(登録後は変更不可)
- ユーザー名
- アイコン画像
- メールアドレス
- パスワード
- 登録日
### DBカラム
- id
- name
- image_path
- mail_address
- password
- created_at

### 表示項目

- 内容

- 誰が書いたか
- 作成日
- 更新日
### DBカラム
- id
- title
- content
- image_path
- author_id
- created_at
- updated_at

- id

- created_at
